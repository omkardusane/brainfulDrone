# Rutvik AORD (Autonomous Object Recognition Drone)

## Rutvik is a Flying machine with a quad-rotor structure which is capable of autonomously executing some paths of flight while collecting and analyzing visuals from onboard camera and its closed loop control system is aided by a gyroscope and accelerometer. It runs Linux and mission control and all hardware control is interfaced with Node.js web server making it a planet wide accessible drone via the internet.
